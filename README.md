# Quick start
## Data
```
cd data
```
- Activate the virtual environment:
```
source bin/activate
```
- Install requirements:
```
pip install -r requirements.txt
```
- Run Flask server on localhost:5000:
```
python app.py
```
## Server
```
cd server
```
- Install dependencies
```
npm i
```
- Run Express on localhost:8080
```
npm start
```
## Client
```
cd client
```
- Install dependencies
```
npm i
```
- Run React client on localhost:3000
```
npm start
```
